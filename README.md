# The BIG POS Typescript SDK

## Installation
Using npm:

```bash
$ npm install @mortgageautomationtechnologies/thebigpos
```

Using yarn:

```bash
$ yarn add @mortgageautomationtechnologies/thebigpos
```

## Quick Start
```js
import TheBigPOS from "@mortgageautomationtechnologies/thebigpos";

TheBigPOS.init();
```
