# The BIG POS Typescript SDK

## Installation
Using npm:

```bash
$ npm install @matech/thebigpos-sdk-typescript
```

Using yarn:

```bash
$ yarn add @matech/thebigpos-sdk-typescript
```

## Quick Start
```js
import TheBigPOS from "@matech/thebigpos-sdk-typescript";

TheBigPOS.init();
```
